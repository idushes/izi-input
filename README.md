# 🎤 Izi Input (macOS Voice Translator)

**Izi Input** — это ультра-легкое, быстрое и полностью автономное macOS-приложение для статус-бара, которое мгновенно переводит русскую речь на английский язык и вставляет текст в любое активное поле ввода. 

Приложение работает на 100% локально и конфиденциально, используя локальную ИИ-модель **Whisper** с аппаратным ускорением **Metal GPU (Apple Silicon)** для моментальной транскрипции.

---

## 🌟 Ключевые особенности / Features

* 🚀 **Metal GPU-ускорение:** Полная оптимизация под чипы Apple Silicon (M1/M2/M3/M4) для мгновенной обработки речи.
* 🔒 **100% Локально и Конфиденциально:** Ни один байт вашего аудио или текста не отправляется в интернет. Работает полностью оффлайн.
* 🌊 **Siri-style Голосовой Оверлей:** При удержании клавиши внизу экрана появляется красивый полупрозрачный индикатор с пульсирующей точкой записи и плавным спектром голосовой волны, танцующим в такт вашему голосу.
* 📋 **Бесшовный автоматический ввод:** Переведенный текст автоматически печатается в активном окне с помощью оптимизированной симуляции нажатий (`Cmd+V`) и безопасного сохранения/восстановления вашего текущего буфера обмена.
* 🎛️ **Выбор ИИ-моделей:** Поддержка моделей разного масштаба (`Tiny` ~75MB, `Base` ~148MB, `Small` ~466MB) с возможностью их скачивания прямо из окна настроек с прогресс-баром.
* 🔋 **Zero-Memory Idle:** В режиме ожидания приложение потребляет минимум ресурсов (~15 МБ ОЗУ) и не нагружает процессор, вызывая Whisper CLI только в момент отпускания клавиши.

---

## 🛠️ Архитектура и Стек / Tech Stack

* **Язык разработки:** Swift, C++ (`whisper.cpp`)
* **Интерфейсы:** SwiftUI (для окна настроек), AppKit/Cocoa (для фоновой работы и статус-бара)
* **Аудио-движок:** `AVFoundation` (запись звука 16kHz Mono PCM WAV, требуемая для Whisper)
* **Keystroke-инъекции:** CoreGraphics (`CGEvent` с использованием `.cgSessionEventTap` для обхода ограничений безопасности macOS)

## 📦 Быстрый старт и Сборка / Build Instructions

Скрипт автоматической сборки берет на себя всю рутину: клонирует `whisper.cpp`, компилирует движок с флагами Metal-ускорения, собирает Swift-исходники, создает и подписывает бандл приложения.

### Шаг 1: Клонирование и Сборка
Выполните в Терминале:
```bash
git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ> izi-input
cd izi-input
chmod +x build.sh
./build.sh
```

### Шаг 2: Запуск приложения
Запустите собранное приложение:
```bash
open IziInput.app
```

## ⚙️ Настройка и Права доступа / Permissions

При первом запуске автоматически откроется окно настроек. Для корректной работы требуются два стандартных разрешения macOS:

1. **Доступ к микрофону:** Запрашивается автоматически при первой записи звука.
2. **Универсальный доступ (Accessibility):** Требуется для симуляции вставки переведенного текста. 
   * В окне настроек приложения нажмите **«Выдать права в настройках macOS»**.
   * В открывшихся системных настройках включите тумблер напротив **IziInput**.
   * *⚠️ Важно: При каждой пересборке приложения (изменении бинарного файла) macOS сбрасывает кэш подписи. Если текст перестал вставляться, просто удалите IziInput из списка Универсального доступа кнопкой «минус» `-` и добавьте его заново!*

## ⌨️ Как пользоваться / How to Use

1. Поставьте курсор мыши в любое текстовое поле (мессенджер, браузер, документ).
2. **Зажмите и удерживайте** клавишу **Fn** (или клавишу с Глобусом 🌐) на клавиатуре.
3. Вы увидите красивую голосовую волну внизу экрана. Начните наговаривать фразу **на русском языке** (например: *«Привет, это работает просто отлично»*).
4. **Отпустите** клавишу **Fn**.
5. Через долю секунды переведенный текст на **английском** автоматически напечатается на месте курсора:
   👉 `Hello, this works just great`
